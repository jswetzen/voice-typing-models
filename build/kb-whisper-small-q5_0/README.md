# kb-whisper-small-q5_0

KB-Whisper Small (q5_0 quantised) — Swedish speech recognition from the National Library of Sweden (KBLab).

- Source: https://huggingface.co/KBLab/kb-whisper-small
- License: Apache 2.0
- WER on FLEURS (sv): 7.3 (vs 7.8 for OpenAI whisper-large-v3, a model 6× larger)
- Size: ~196 MB
